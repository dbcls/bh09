BioMart - Data Integration and Interoperability

BioMart is a data-management and integration system for large scale biological data.

Project page:  http://www.biomart.org

Team

  • Arek Kasprzyk
  • Syed Haider

Goals

  • To collaborate with data visualisation & workflow system writers towards better integration of BioMart resources over the web services.

Summary

1- Collaborated with Galaxy, GBrowse and Cytoscape developers for an improved BioMart Client to access BioMart databases using BioMart web services.

2- Hands-on session on how to use BioMart and which databases are available on BioMart Central Portal  http://www.biomart.org/biomart/martview.
